# House of Fraser: Data-Driven Business Failure Analysis

## Overview

This project investigates the decline and acquisition of House of Fraser using financial, customer, and market data.

The analysis explores:

- Branding and marketing failures
- Management and ownership instability
- Financial performance decline
- Customer perception and market share trends

Tools used include Python, Excel, regression analysis, and data visualization.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook
- Excel

---

## Key Insights

- Weak brand identity negatively impacted competitiveness
- Delayed e-commerce adoption hurt market position
- Management instability disrupted long-term strategy
- Operating costs and debt reduced profitability
